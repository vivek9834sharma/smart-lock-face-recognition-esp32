# 🔐 Smart Lock System using Face Recognition & ESP32

## 📌 Overview
An IoT-based smart lock system that uses Face Recognition for secure authentication and ESP32 microcontroller for hardware control.

This project eliminates traditional key-based systems by implementing AI-powered facial verification.

---

## 🚀 Features
- Face-based authentication
- Real-time face detection and recognition
- ESP32 hardware integration
- Automated door locking/unlocking mechanism
- Secure access control system

---

## 🛠️ Tech Stack
- Python
- OpenCV
- Face Recognition
- ESP32
- IoT

---

## 🏗️ System Architecture
1. Camera captures live image
2. Face recognition model verifies identity
3. If match found → Signal sent to ESP32
4. ESP32 activates locking mechanism
5. If no match → Access denied
